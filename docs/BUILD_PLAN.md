# VISR Build Plan

## Milestone 0 — Repository Foundation

- Next.js App Router scaffold
- TypeScript strict mode
- VISR design tokens
- baseline metadata
- loading, error, and 404 states
- Vercel Analytics and Speed Insights
- Experience Bible stored in source control

## Milestone 1 — Opening and Reveal

- production opening director
- adaptive poster/video system
- acrylic edge-light reveal
- first-visit and returning-visit behavior
- skip intro and reduced-motion fallback

## Milestone 2 — Presentation and Engineering

- presentation transformation
- engineering timeline
- exploded-view media adapter
- technical notation system
- material-specific sound layer

## Milestone 3 — Halo Collection

- six Halo material states
- desktop preview/select behavior
- mobile swipe behavior
- persistent selected Halo state
- Halo product content model

## Milestone 4 — VISR Carry

- Arrival cinematic sequence
- Carry product page
- lifestyle media pipeline
- transition from selected Halo into Carry

## Milestone 5 — Exhibition

- collector-story content model
- editorial gallery
- image optimization pipeline
- collector/location attribution

## Milestone 6 — Commerce Foundation

- Neon PostgreSQL
- Drizzle schema and migrations
- products, variants, inventory, carts, orders
- guest cart and server validation
- operations access foundation

## Milestone 7 — Configuration and Checkout

- VISR configurator
- Halo compatibility
- Carry extension
- checkout and address validation
- Midtrans sandbox integration

## Milestone 8 — Shipping and Operations

- Biteship rate selection
- automatic AWB
- shipment webhooks
- transactional notifications
- operations panel and retries

## Milestone 9 — Validation and Launch

- Playwright critical journeys
- accessibility review
- visual regression
- performance budgets
- analytics funnel validation
- preview approval and production release
